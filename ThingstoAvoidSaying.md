**Never Say → Say Instead**

❌ Platform/Technology Gap

Never say:

"I haven't worked with Databricks / Snowflake / Vertex AI / [any platform] so I can't speak to that."

Say instead:

"I haven't used [X] specifically, but the problem it solves is [Y]. I've addressed that using [Z]. The architectural principles are the same — [concept]. I'd be productive on it quickly."


❌ Uncertainty

Never say:

"I'm not sure / I don't know / That's a good question, I'll have to think about that" — as a complete answer.

Say instead:
"I want to think through that properly rather than give you a half-answer. My initial instinct is [X] — but I'd want to validate that against [Y]. Can I follow up on the exact figure?"


❌ Tool listing without justification

Never say:
"We used GPT-4 / we used LangChain / we used Pinecone."

Say instead:
"We chose GPT-4 over open-source alternatives because [specific reason — latency, quality, data sensitivity, cost]. The trade-off we accepted was [X]. We mitigated that by [Y]."

❌ Vague outcomes

Never say:
"It improved performance significantly / we got great results / the client was happy."
Say instead:
"It reduced latency from 1.8 seconds to 340ms / improved precision@5 from 0.71 to 0.89 / reduced hallucination rate from 8% to 1.1% — which unblocked [specific business outcome]."

❌ Claiming best practices without naming them

Never say:
"We followed best practices."
Say instead:
"We implemented RAGAS evaluation on every deployment with a faithfulness threshold gate at 0.75, and LangSmith for per-step tracing of agent runs."

❌ Unproven learning speed claims

Never say:
"I'm a quick learner / I pick things up fast / I'm adaptable."

Say instead:
"I came up to speed on [specific technology] in [specific timeframe] and [specific outcome]. For example, [one concrete learning story]."


❌ Team credit without ownership clarity

Never say:

"My team did X."

Say instead:


"I designed [the architecture / evaluation framework / data pipeline]. My team implemented it. I was responsible for [specific technical decisions] and mentored [number] engineers through [specific aspect]."


❌ Excessive politeness filler

Never say:

"That's a really great question!" — every time a question is asked.

Say instead:
Just answer. If you need a moment: "Let me make sure I answer that precisely." One brief pause, then a sharp answer.
